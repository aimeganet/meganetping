# 🚀 MegaNet Ping Monitor

أداة بسيطة وقوية لمراقبة اتصال الشبكة (Ping Monitor) تقوم بفحص حالة الخوادم والأجهزة على الشبكة ومعرفة زمن الاستجابة (Latency) واستقرار الاتصال.

تم تطوير هذا المشروع لتسهيل مراقبة الشبكات واكتشاف المشاكل بسرعة، وهو مناسب لمهندسي الشبكات ومديري الأنظمة.

---

## 📡 مميزات المشروع

- فحص اتصال الشبكة باستخدام **Ping**
- قياس زمن الاستجابة **Latency**
- مراقبة استقرار الاتصال
- دعم اختبار عدة عناوين IP أو Domains
- أداة خفيفة وسريعة الاستخدام
- مناسبة لاختبار الشبكات الداخلية والخوادم

---

## 🛠 التقنيات المستخدمة

- Python / Node.js
- ICMP Ping
- أدوات تشخيص الشبكات
